- ENGLISH

Simple Password Generator

Portable version for Windows.

How to use:

- Copy this .exe file to a USB drive.
- Open the executable on any Windows computer with .NET Framework 4.x.
- The app does not require installation and does not ask for administrator permission.

Security note:

- The executable was compiled locally from the PasswordGeneratorPortable.cs file.
- It does not use a packager, obfuscator, installer, or dependency download.
- Because there is no digital publisher signature, Windows SmartScreen may display an "unknown publisher" warning on some computers.
- Enjoy!
